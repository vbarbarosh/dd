Cancellation and rollback

What it is: A drag can be canceled with Escape (or when the window loses focus). The end handler checks ctx.canceled and restores the square's starting position.

How to use it:

  1. Drag a square to a new position and release normally; it stays there.
  2. Drag it again, keep holding the pointer, and press Escape.
  3. The canceled drag ends immediately and the square jumps back to where that drag began.
