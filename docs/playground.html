Build a drag, then inspect every event.

Change the options, drag the block, and copy the generated setup. Everything here uses the same dist/dd.js shipped by the package.

What is this?

A no-build sandbox for combining dd options and seeing the context object change as real pointer events arrive.

How to use it

Choose a threshold or grid, drag the blue block, and watch the event list. Press Escape to test cancellation; copy the generated code when the behavior feels right.

Tip: with a threshold selected, the block is “armed” until you move far enough. Press Escape during a drag to cancel it.

    Translated coordinates survive scrolling

    Drag the orange dot, then use a wheel or trackpad over this viewport without releasing it. translate converts client coordinates into the scrollable world's coordinate system.

    World position: 120,100

    Local coordinates survive CSS transforms

    This world is rotated and scaled with CSS. Its translate handler applies the inverse transform, so the purple marker still moves in the world's untransformed coordinate system.

    Local world position: 180,120

    Handler order at a glance

    Mixins run first, then your context handler. A threshold suppresses the regular handlers until it is crossed; the *_nothreshold hooks still run.

    1. 1
      begin_nothreshold
      Immediately on press
    2. 2
      begin
      Immediately, or after threshold
    3. 3
      move → update
      For each pointer move
    4. 4
      end → end_nothreshold
      On release or cancellation

    Try it above and watch the event history. On cancellation, ctx.canceled becomes true.
