Grid snapping

What it is: The dd.grid(50) mixin rounds pointer coordinates to the nearest 50-pixel step. This is useful for tile editors, diagram nodes, and aligned layouts.

How to use it:

  1. Drag any square slowly in any direction.
  2. Notice that its displayed coordinates change in 50-pixel increments.
  3. Release between grid lines; the square remains on the nearest grid point.
